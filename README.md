# Sistema de Gestión de Tienda - Java

Aplicación desarrollada en Java como proyecto académico del ciclo DAM (Desarrollo de Aplicaciones Multiplataforma).

## Descripción

Sistema de gestión de tablets que permite:

- Dar de alta productos
- Modificar datos
- Gestionar stock
- Vender tablets a clientes
- Generar facturas
- Mostrar productos disponibles y vendidos

Aplicación desarrollada en consola aplicando Programación Orientada a Objetos.

## Tecnologías utilizadas

- Java
- Programación Orientada a Objetos
- Clases y objetos
- Separación por capas
- Lógica de negocio
- Validación de datos

## Estructura

- `Main.java` → Punto de entrada
- `GestionTienda.java` → Lógica principal del sistema
- `Tablet.java` → Modelo de datos
- `Cliente.java` → Gestión de clientes

## Conceptos aplicados

- Encapsulamiento
- Modularidad
- Estructuras de datos
- Control de flujo
- Gestión de estado

Proyecto desarrollado como práctica académica.
